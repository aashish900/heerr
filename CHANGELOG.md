## 2026-06-08 ‚Äî B2 auth dependency

- `backend/app/api/__init__.py`, `backend/app/api/deps.py`.
- `bearer_token` (FastAPI dep): reads `Authorization: Bearer ‚Ä¶` via `HTTPBearer(auto_error=False)`; rejects missing/wrong-scheme/unknown/revoked with `401` + `WWW-Authenticate: Bearer` header; looks up by `sha256(raw)` hash; returns the `Token` ORM row.
- `require_scope(*scopes)` (factory): `403` when the required scopes aren't a subset of `tok.scopes`; otherwise returns the token.
- `require_admin` (dep): `403` unless `tok.is_admin`; chains on `bearer_token` so unauthenticated admin calls still surface `401`.
- `backend/tests/test_auth.py` ‚Äî 12 tests across the full state machine: no-header / wrong-scheme / unknown / revoked ‚Üí 401; happy path returns owner+scopes; single-scope missing ‚Üí 403; single-scope present ‚Üí 200; multi-scope partial ‚Üí 403, full ‚Üí 200; non-admin on admin ‚Üí 403; admin on admin ‚Üí 200; admin route with no auth ‚Üí 401.
- Test fixtures (in `test_auth.py`): `app_engine` / `app_sm` (function-scoped, derived from `pg_async_url`); `auth_app` (FastAPI test app with `get_session` overridden + three sample routes); `client` (`httpx.AsyncClient` over `ASGITransport`); `make_token` (factory yielding raw tokens and cleaning up rows on teardown).
